Для работы необходимо установить node.js
для запуска в консоли: 
1. запустите консоль
2. перейдите в директорию с index.js
3. запустите введя: node index аргумент
аргумент римское или арабское число для конвертации 

Корректно конвертируются целые числа от 1 до 3 999 (от I до MMMCMXCIX).
Автоматически определяет римское или арабское число было введено и конвертирует его.
